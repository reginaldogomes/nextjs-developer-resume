O App Router introduz conceitos como Server Components e layouts aninhados.

### Vantagens

1.  **Layouts Simplificados**: Cada pasta pode ter seu próprio layout.
2.  **Server Components por Padrão**: Melhor performance, pois o código roda no servidor.
